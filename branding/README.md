# Teller Branding Assets

This directory contains the official Teller brand identity package provided by the graphics designer.

## Contents

- **Teller-IdentityPackage-2020/** - Complete Teller brand package
  - Logo variations (ForOnWhite, ForOnDark, OnDark)
  - Icon variations
  - TellerStyleGuide-2020.pdf - Official brand guidelines
  - Logo files in multiple formats (.svg, .png, .tif)

- **CanAmLogoPackage/** - CanAm company branding
  - CanAm is the company that owns the Teller product

## Usage

The branding assets in this folder are **excluded from git** due to large file sizes (TIF files and PDFs).

Only the necessary SVG logo has been copied to the frontend:
- `frontend/public/teller-logo.svg` - Used in the application

## Brand Colors

From TellerStyleGuide-2020.pdf:

### Neutrals
- Charcoal: `#494D50`
- Dark Silver: `#A5A5A5`
- Silver: `#E6E6E6`

### Complementary
